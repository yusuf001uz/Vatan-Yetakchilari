# Vatanimni Bilaman BOT 🇺🇿

O‘zbekiston tarixi, madaniyati va buyuk siymolari haqidagi bilimlarni boyitishga xizmat qiluvchi intellektual Telegram platformasi.

## ✨ Imkoniyatlar
* **AI Yordamchi:** Botga o‘rnatilgan sun’iy intellekt bilan istalgan mavzuda muloqot qilish.
* **Intellektual Viktorinalar:** Tarixiy voqealar va shaxslar bo‘yicha testlar orqali bilimni sinash.
* **O‘quv Kurslari:** Milliy tariximizni tizimli o‘rganish uchun maxsus darslar.
* **Reyting Tizimi:** Foydalanuvchilar orasida bilim darajasi bo‘yicha yuqori o‘rinlar uchun kurash.

## 🚀 Texnologiyalar
* **Python 3.x**
* **aiogram** — Framework
* **SQLite / PostgreSQL** — Database

## 🛠 O'rnatish

git clone [https://github.com/username/vatanimni-bilaman.git](https://github.com/username/vatanimni-bilaman.git)
cd vatanimni-bilaman
pip install -r requirements.txt
python main.py
